Projects

MovieBooking

The MERN Stack Movie Booking Application simplifies the process of online movie reservations for both administrators and users. Administrators benefit from a secure login with email/password, granting them complete control over movie-show management, including addition, editing, deletion, and disabling. On the user side, a user-friendly sign-in feature allows seamless exploration of available shows on a specified date, enabling easy ticket booking. Users receive confirmation screens and email notifications upon successful bookings, and the "My Bookings" page provides access to a history of previous reservations. The application's PDF ticket download feature, incorporating a QR code with show details. Developed with React for the frontend and Node.js, Express.js, and MongoDB for the backend, this MERN Stack application prioritises simplicity and functionality, catering to the needs of both administrators and movie enthusiasts.

ShortenUrl

ShortenURL is a user-friendly project developed with Laravel, MySQL, and blade.php that allows users to register and log in. Once logged in, users can easily shorten long URLs, making them more manageable. The project provides the convenience of editing and deleting entries, giving users control over their shortened links. With a secure login system and intuitive interface, ShortenURL offers a simple yet effective solution for URL management.

MedicalStore

The Medical Store Management System using Express simplifies medicine management with functionalities such as listing, adding, viewing, editing, and deleting medicines. It uses search functionality for efficient navigation. The system provides a user-friendly interface for managing medicines, presenting a list with essential details like quantity, price, and expiration date. Users can seamlessly add new medicines, view and edit their details, delete entries, and utilise the search feature. This Express-based system ensures efficient medicine inventory management, catering to the needs of medical store operations.
